# RunRec MVP

RunRec é um MVP interativo para recomendação contextual do próximo treino de corrida.

O app combina:

- objetivo do atleta;
- volume das últimas semanas;
- pace confortável;
- dias desde o último treino intenso;
- status atual de corrida;
- estado estimado de fadiga;
- regras de segurança.

A recomendação final é apresentada como um ranking de próximos treinos possíveis, com explicação da decisão.

## Como executar localmente

Instale as dependências:

```bash
pip install -r requirements.txt
```

Rode o app:

```bash
app/app.py
```

Observação
Este app é uma demonstração educacional do projeto de capstone. Ele não substitui avaliação médica, treinador profissional ou plano individualizado de corrida.
